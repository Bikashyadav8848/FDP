```bash

```

```
http://localhost:8000
```

### 4. Explore Each Module
- Click on each module in the navigation
- Try adding, editing, and deleting records
- Test filtering and search
- View statistics

### 5. Use the Admin Panel
```
http://localhost:8000/admin/
```

## 🎯 What Each Module Does

### Students
Track all student information including name, email, and roll number.

### Teachers
Manage teacher records and assign them to subjects.

### Classes
Organize students into class sections.

### Subjects
Create subjects, assign subject codes, and link to teachers.

### Attendance ⭐
Mark daily attendance, filter by student/date, view statistics.

### Timetable ⭐
Create and view weekly class schedules with subjects and teachers.

### Fees ⭐
Manage fee categories, structures, student assignments, and payments with automatic balance calculation.

### Marks
Record and track student grades.
